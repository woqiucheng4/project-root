# 🌐 自动化爬虫引擎与数据巡检工作流配置指南

本系统包含一套工业级的**无人值守每日考情数据同步引擎**。它通过集成到代码仓库的云环境（例如 GitHub Actions），每日自动监控、爬取、洗脱数据并合并入库，整个过程不需要人为干预。

---

## 🏗 一、核心脚本构成与运行原理

爬虫及数据处理矩阵分布在 `scripts/` 目录下，按以下顺序串联形成闭环：

### 1️⃣ 捡漏情报探测雷达 (`scripts/crawl-daily-heat.js`)
*   **功能**：在短如十天的极速出档报志愿期期间，每日自动打捞出当前整个基本面被冷落（如一直保持 0 人报考、或者条件极其诡异无人填报）的高容错神仙岗位，产出 `data/daily-leak-jobs.json`。
*   **应用**：此结果文件直接被前端的 `<LeakBanner />` 动态拉取，在主页上营造出极具压迫感和实效性的“实时赛况走马灯墙”。
*   **如何手动运行**：
    ```bash
    node scripts/crawl-daily-heat.js
    ```

### 2️⃣ 全网省考/国考新源监听探针 (`scripts/auto-discover-provincial.js`)
*   **功能**：应对因各省级政务平台高度分散导致的数据爬取断层难题。它利用无头网络框架，通过轮询大型教培汇总机构页面或搜索引擎，侦测新公布（诸如“江苏省2026年公务员职位表.xls”）的最新报表源。
*   **特点**：一旦发现目标，立刻下载并将其实体 Excel 落盘入库；并进一步**递归唤醒**下面的解析清洗工具。
*   **如何手动运行**：
    ```bash
    node scripts/auto-discover-provincial.js
    ```

### 3️⃣ 底层清洗与护城河挖掘流水线 (`scripts/parse-jobs.js` & `enhance-jobs.js`)
*   **功能**：前者提取并丢弃无用结构行，标准化成 `jobs-parsed.json`；后者利用自然语言正则表达式（NLP Regex）进行二次穿透，挖出岗位的 `要求四六级证书金额`、`法考/注册会计师金矿证书`、`地区及特殊性别歧视红利`。且还会逆推还原该岗位基于当年度整体内卷值的虚拟历史进面分数。
*   **组合运行方式**：可由 `auto-discover-provincial.js` 在探测成功后自动化内联调用。

## 🚀 二、如何将它完全“挂机”？(配置 GitHub Actions 定时任务)

系统默认配备了一个非常完善的 `cron` 定时任务描述文件： `.github/workflows/daily-data-sync.yml`。
它会租用云端最新的服务器，在每天**凌晨 03:30** 完全自动化执行以上三步。

### **如何使其生效？**

1.  **提交至 GitHub**：将当前整个 `kaogong` 目录关联并推送到你的个人 GitHub 仓库。
2.  **调整你的云推密钥（GITHUB_TOKEN 权限）**：
    通常你无需额外配置环境变量，但为了让你的 Workflow 执行机器人在爬取到新数据后能够“自己提交一份 commit 并把新表塞回大仓”，你必须在 GitHub 所在仓库的配置页 `Settings -> Actions -> General -> Workflow permissions` 下，选中 **`Read and write permissions`**（允许读写权）。
3.  **补充微信服务号推送密钥 (可选)**：
    为了在完成抓取后自动通过“模板消息”震撼到每个用户的手机微信面板上，你需要：
    在 GitHub 该仓库的 `Settings -> Secrets and variables -> Actions` 中，新建两个 Secret：
    *   `WX_APP_ID`: (填入你在公众号平台的服务号 ID)
    *   `WX_APP_SECRET`: (填入此服务号开发者密码)

### **发生变动了会怎么样？**

当夜间脚本侦测并产生新数据（哪怕只是新出了5个没人报的捡漏职位）并推送到仓库的 `main` 分支后：
由于本系统通常部署在 **Vercel** 平台上，Vercel 只要嗅探到 `main` 分支有更新，就会被自动唤醒并执行一次长达几秒的重编译热加载。
**也就意味着：天亮当用户一觉醒来打开网页时，他们看到的，已经是整个公考圈目前最新鲜热辣的一手前线数据。你实现了无需开电脑的挂机赚被动收入架构。**

## 🚑 三、灾备与紧急干预法则

有极罕见的情况（比如某省级政务网的职位表改成了无法处理的 PDF 扫描图），导致自动探测器 `auto-discover-provincial.js` 彻底罢工或者崩溃。
此时你可以采取 **“高级人工干预工作流 (import-job-data)”**：

1.  手工利用 QQ 群或找辅导机构拿到该省考的原始 Excel 文件（如 `2026-sz.xls`）。
2.  将其直接强行拖入你电脑的本地目录：`kaogong/data/raw/` 中。
3.  运行命令 `node scripts/parse-jobs.js data/raw/2026-sz.xls`。
4.  运行命令 `node scripts/enhance-jobs.js`。
5.  提交代码 `git psuh`，大仓重置完毕。
